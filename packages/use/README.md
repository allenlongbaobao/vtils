## 安装

```bash
# yarn
yarn add @vtils/use

# or, npm
npm i @vtils/use --save
```

你也可通过 CDN 安装，然后使用全局变量 `vhooks` 访问相关工具：

```html
<script src="https://cdn.jsdelivr.net/npm/@vtils/use@2.27.0/lib/index.umd.min.js" crossorigin="anonymous"></script>
```
